Raspberry Pi Grundkonfiguration

File /boot/cmdline.txt, insert: console=serial0,115200 console=tty1 root=PARTUUID=b3072c18-02 rootfstype=ext4 fsck.repair=yes rootwait quiet splash plymouth.ignore-serial-consoles ip=10.1.2.206::10.1.2.1:255.255.255.0:rpi4-Master:eth0:off cgroup_memory=1 cgroup_enable=memory

Installiere Salt Minion: sudo apt-get install salt-minion

Aktiviere RDP (wenn GUI am Pi aktiv)

Setze root Password: passwd root

Suche folgenden Eintrag "#PermitRootLogin without-password" in sudo nano /etc/ssh/sshd_config und entfernen den Kommentar # und setze: PermitRootLogin yes
